Knowledge Base

To set up your class roster:

  1. Navigate to the "Classes" section in your dashboard
  2. Click "Create New Class" or select an existing class
  3. Use the "Add Students" button to search for and add students
  4. You can import students via CSV for bulk additions
  5. Save your changes when complete

Pro Tip

Use the student ID numbers for quick identification when managing large classes.

The gradebook allows multiple customization options:

  • Weighted categories (tests, homework, participation)
  • Curved grading scales
  • Dropping lowest scores
  • Extra credit assignments

To access these settings:

Go to Gradebook → Settings → Grading Rules

Important

Changes to grading rules will affect all existing calculations. Review carefully before saving.

If you're having trouble accessing your account:

Forgot Password

Use the "Forgot Password" link on the login page to reset your credentials.

Account Locked

After multiple failed attempts, wait 15 minutes or contact support.

Browser Issues

Try clearing cache/cookies or using a different browser.
